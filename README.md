## Features

### Virtual Variables and Lists
Virtual Variables and Virtual Lists let you create variables and lists in realtime!

### Logic and Checks
Check if Normal or Virtual Variables and lists exist and more!

### Saves (Must be unsandboxed to use this feature)
This lets you create a save using blocks which lets you carry data even after reload!
